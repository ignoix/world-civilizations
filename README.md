# World Civilizations 世界文明時間線

一個純前端、可縮放拖曳的互動式歷史時間線，涵蓋亞洲、中東、歐洲多個文明的朝代、君主與重大歷史事件，並整合 Wikipedia 摘要與圖片。

## 使用方式

直接用瀏覽器打開 [index.html](index.html) 即可，無需安裝或建置。

- 滑鼠拖曳／滾輪縮放（Ctrl + 滾輪）瀏覽時間軸
- 頂部分組篩選列可切換顯示各文明區段
- 點擊朝代／君主色塊可查看 Wikipedia 簡介與地圖

## 涵蓋內容

- **亞洲**：中國歷代（[zh/](zh/)）、蒙古帝國、古印度
- **中東**：埃及、巴比倫、亞述、赫梯、波斯、奧斯曼帝國
- **歐洲**：古希臘、馬其頓、羅馬帝國、英國、法國、德國
- 其他：[聖經歷史](bible/)、[摩門經歷史](mormen/)

## 專案結構

每個文明一個資料夾，內含 `*.js` 資料檔（朝代區間、君主列表、事件），由 [index.html](index.html) 統一載入並渲染。
